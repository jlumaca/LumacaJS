# 🚗 BuyCars UY - Gestión de Autos

Aplicación web simple para gestionar autos (CRUD) con sistema de usuarios.  
Desarrollada con **Node.js (Express)** en el backend y **HTML + JS** en el frontend.

---

## 📌 Funcionalidades

### 👤 Usuarios
- Registro de usuarios
- Login
- Logout

### 🚘 Autos
- Agregar autos
- Listar autos
- Buscar autos (por código, marca o modelo)
- Modificar autos
- Eliminar autos
- Subir imagen de autos (con `multer`)

---

## 🛠️ Tecnologías utilizadas

- Node.js
- Express
- Multer (subida de imágenes)
- JavaScript (Frontend)
- HTML + Bootstrap
- JSON como almacenamiento (simulación de base de datos)
